# API & Integration Guide

This project provides a professional **RESTful API** documented with **Swagger (OpenAPI)**. The API supports both Web and Flutter clients with high-performance gesture recognition and real-time chat capabilities.

## 🚀 API Documentation (Swagger)

The primary source of truth for all endpoints, request formats, and response schemas is the Swagger UI.

### Live Documentation
When the server is running, access the interactive docs at:
**[http://localhost:3000/api-docs](http://localhost:3000/api-docs)**

### Specification File
Alternatively, you can find the static OpenAPI specification here:
**[API/swagger.json](./swagger.json)**
